# Electrical Receptacle Symbol Detector

Automatically detect and count electrical receptacle symbols in architectural drawings.

## Features

- 🎯 **Automated Symbol Extraction**: Extracts receptacle symbols from legend pages
- 🔍 **Multi-Scale Detection**: Finds symbols at different scales and rotations
- 🎨 **Visual Results**: Generates annotated images with detected receptacles
- 📊 **JSON Output**: Saves detection coordinates and confidence scores
- 🔄 **Multiple Detection
